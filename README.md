# ВЕЛЬВЕТ — лендинг салона красоты

Одностраничный сайт для салона красоты полного цикла. Цель — заявка на онлайн-запись.
Сделано как портфолио-проект по ТЗ. Данные салона вымышленные (демо для показа заказчикам).

## ✨ Что внутри

- **9 блоков**: шапка, герой, акции, категории услуг, прайс (табы), о нас, отзывы, контакты + форма записи, футер
- **Sticky-шапка** с уплотнением при скролле, **бургер-меню** (оверлей, закрытие по фону / Esc / клику)
- **Плавный скролл** по якорям, **scroll-reveal** появление секций (IntersectionObserver)
- **Прайс с переключением направлений**, категории кликом ведут в нужную вкладку
- **Форма записи** с валидацией (имя, телефон, согласие) и экраном «Заявка принята» (отправка-заглушка)
- **Адаптив** mobile-first, SEO-метаданные (title, description, OpenGraph), уважение `prefers-reduced-motion`

## 🛠 Стек

| Слой | Технология |
|---|---|
| Разметка | Чистый HTML5, семантика |
| Стили | Нативный CSS (переменные, Grid, Flexbox) |
| Логика | Ванильный JavaScript |
| Шрифты | Системные (Georgia + system-ui), без CDN |
| Картинки | CSS-градиенты и inline-SVG, без внешних хостов |
| Зависимости | Нет. Ноль внешних библиотек и запросов |

Один файл `index.html` со встроенными `<style>` и `<script>`. Открывается двойным кликом даже без интернета.

## 🚀 Запуск

Никакой сборки не требуется:

```bash
# просто откройте index.html в браузере
# или поднимите статику локально:
npx serve .
```

## 🔌 Точки расширения под боевую версию

В портфолио-версии всё работает автономно. Места для подключения под реального заказчика:

- **Отправка заявки** → сейчас заглушка в `<script>` (функция submit). Подключается webhook CRM (AmoCRM / Bitrix24) или Telegram-бот / сервис онлайн-записи (YCLIENTS).
- **Аналитика** (Я.Метрика / GA4) → добавляется в `<head>`.
- **Контент** (услуги, цены, контакты, соцсети) → правится прямо в разметке; палитра и шрифты вынесены в CSS-переменные в начале `<style>`.
